## 📌 Project Summary

Developed a scalable Disaster Management System using the MERN stack to streamline incident reporting, emergency response coordination, and volunteer management.

- Implemented secure authentication using JWT  
- Designed RESTful APIs for efficient communication  
- Built responsive UI for real-time updates  
- Enabled coordination between users, volunteers, and authorities

## ⚙️ Installation & Setup

### 1. Clone Repository

git clone https://github.com/narutopandit/Disaster-Management-System.git

cd Disaster-Management-System


### 2. Install Dependencies

#### Backend

cd server
npm install


#### Frontend

cd frontend
npm install


---

### 3. Environment Variables

Create a `.env` file in the server folder:


PORT=
MONGO_URI="mongodb+srv:.mongodb.net/"
JWT_SECRET=''
JWT_REFRESH_SECRET=''
GNEWS_API_KEY=''
SYSTEM_USER_ID=''

Create a `.env` file in the server folder:

VITE_SOCKET_URL=
VITE_API_URL = 

---

### 4. Run the Application

#### Backend

npm start


#### Frontend

npm run dev
